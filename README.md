## 项目结构

```bash
├── data/            # 数据文件与预处理脚本
├── src/             # 主程序代码（前后端）
├── assets/          # 图片、图标等资源
├── docs/            # 项目说明文档
├── README.md        # 项目说明文件
```

## 📊 技术栈

- 前端：Vue.js + ECharts
- 后端：Flask / Node.js / FastAPI
- 数据处理：Pandas / NumPy
- 数据来源：ChinaVis 2025 官方比赛数据集
